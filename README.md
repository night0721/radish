# Radish
Programming language.

# Usage
```sh
rd run # interpreter
rd build # compiler
rd add # dependency manager
rd test # test suite
rd ci # ci
rd lsp # lsp server
rd packager # create package
```

# Depedencies
None

# Building
You will need to run these with elevated privilages.
```
$ make
# make install
```
